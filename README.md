# 💸 FinSight Dashboard

A modern, responsive finance dashboard built using React, focused on clean UI, smooth user experience, and scalable frontend architecture.

---

## 🚀 Live Demo
👉 [FinSight](https://fin-sight-gamma-ten.vercel.app/)

---

## 📌 Overview

FinSight is a frontend-focused finance dashboard designed to demonstrate strong UI/UX design, component structuring, and state management.

The goal of this project was to build a clean and interactive dashboard interface that simulates real-world financial tracking applications.

---

## 🧠 Approach

The project was built with a focus on:

- **Component-based architecture**  
  UI is broken down into reusable components like Sidebar, Layout, and Cards.

- **State management using Zustand**  
  Used for managing global UI state such as sidebar toggle, theme switching, and role selection.

- **Scalable folder structure**  
  Organized into `components`, `pages`, `stores`, and `hooks` for maintainability.

- **Responsive design**  
  Ensures usability across desktop and mobile devices.

- **Modern UI practices**  
  Includes smooth animations, consistent spacing, and accessible color themes.
